NyvoStudio algorithm writing style guide :

    1. Loop starting , Finishing , Change In operations will be written In sqaure.

    2. Normal logical activites will be written In circle ( Not classes , Functions , Decorators )

    3. User Inputs are shown in Hexagon shape.

    4. System Decisions And exceptions will be written In rhombuse.

    5. Functions Will be written in triangles.

    6. Function Calls be written In Call Activity Shape ( Yellow Square ).

    7. Connected attributes will be written In Page refrences.

    8. Database and class connections will be written In Database Shape.

    9. Connections will be Refrenced With circlehead Lines.

    10. Imports and accesses will be Refrenced With Arrowhead Lines.

    11. Partitions will be shown In vertical Lines.

    12. Partition Names will be shown in Horizontal Lane.

    13. Comments will be sown In Comment_Alt shape.

    14. Orderings Will be Written In brackets ( Number ) .

    15. All algorithms Must be written In visual paradigm online.

NyvoStudio filing style guide :

    1. All file name's should be in snake_case : ( All letter's are in lowercase , Instead of space _ Is used. ) . 

    2. File name's should start With a Verb , And at the second part's there should be A name or piece of information.

    3. File name's should be descriptive And self explanatory .

    4. Do not use Complex filename's , All name's should be kept simple.
